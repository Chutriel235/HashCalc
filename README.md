# HashCalc by jNizM
Calculate hash from string via AutoHotkey


## Features
* Generate hash value from string
* Secure-Salted function
* Verify hash

### Hash Functions
* MD2
* MD4
* MD5
* SHA-1
* SHA-256
* SHA-384
* SHA-512
