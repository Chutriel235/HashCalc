# HashCalc by jNizM
Calculate hash from string or file via AutoHotkey


## Features
* Generate hash value from string or file
* Secure-Salted function
* Verify hash

### Hash Functions
* MD2 (+ HMAC)
* MD4 (+ HMAC)
* MD5 (+ HMAC)
* SHA-1 (+ HMAC)
* SHA-256 (+ HMAC)
* SHA-384 (+ HMAC)
* SHA-512 (+ HMAC)
